# get-WindowsTerminalSettingsPath

## SYNOPSIS

Gets the windows terminal settings path.

## SYNTAX

```powershell
get-WindowsTerminalSettingsPath
```

## DESCRIPTION

If Windows terminal is not installed, this file won't exist, so
empty string is returned.

## PARAMETERS

## INPUTS

### None

## OUTPUTS

### System.String
