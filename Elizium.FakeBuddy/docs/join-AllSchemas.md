# join-AllSchemas

## SYNOPSIS

Builds the json content representing all the schemes previously collated.

## SYNTAX

```powershell
join-AllSchemas [[-Schemes] <Hashtable>] [<CommonParameters>]
```

## DESCRIPTION

Used by ConvertFrom-ItermColors.

## PARAMETERS
